# Security & Legal Review Artifacts

This folder contains high-level artifacts for security/legal review of a whistleblower submission platform.

## Files
- `legal-demand-handling-sequence.mmd` — Mermaid sequence diagram source.
- `whistleblower-policy-redline.md` — Redline-friendly policy draft with `[COUNSEL NOTE]` clauses.

## Generate PNG locally
If Mermaid CLI is available:

```bash
mmdc -i docs/security/legal-demand-handling-sequence.mmd -o docs/security/legal-demand-handling-sequence.png
```
